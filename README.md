# Shutty Utility

> A lightweight Windows utility for scheduling shutdown, restart, and sleep actions.

---

## 🇷🇺 Русский

**Shutty** — простая утилита для Windows, которая позволяет настроить таймер выключения, перезагрузки или перехода в режим сна.

### Возможности
- Таймер выключения, перезагрузки и сна
- Минималистичный интерфейс

### Требования
- Windows 10 / 11
- [.NET 8 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

### Установка
Скачайте последний релиз со страницы [Releases](https://github.com/jeanofkeep/shutty_utility_win/releases) и запустите исполняемый файл — установка не требуется.

---

## 🇬🇧 English

**Shutty** is a simple Windows utility that lets you schedule shutdown, restart, or sleep with a timer.

### Features
- Shutdown, restart, and sleep timer
- Minimal and clean UI

### Requirements
- Windows 10 / 11
- [.NET 8 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

### Installation
Download the latest release from the [Releases](https://github.com/jeanofkeep/shutty_utility_win/releases) page and run the executable — no installation required.
